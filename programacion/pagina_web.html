Desarrollo y Personalización de Página Web Minimalista

Este proyecto comenzó como un ejercicio experimental con un template básico de Bootstrap y evolucionó hacia la creación de una página web propia, funcional y moderna. El objetivo fue convertir una base genérica en un espacio digital único que reflejara mi estilo y me permitiera centralizar diferentes áreas de interés.

El trabajo implicó rediseñar la estructura, adaptar componentes y aplicar estilos personalizados para lograr una experiencia visual minimalista pero impactante. Se trabajó con un enfoque “mobile-first”, asegurando compatibilidad y fluidez en todo tipo de dispositivos.

Durante el proceso se integraron librerías como SwiperJS para sliders dinámicos y AOS para animaciones de entrada, optimizando también el rendimiento mediante compresión de imágenes y carga diferida.

Más allá del desarrollo técnico, esta experiencia reforzó mis habilidades en diseño UI/UX, gestión de recursos y organización de contenidos, entendiendo cómo una interfaz bien pensada mejora la interacción del usuario.

Este proyecto se convirtió en un laboratorio personal de pruebas para nuevas tecnologías y técnicas de diseño, sentando las bases para futuros desarrollos web más complejos y totalmente personalizados.
